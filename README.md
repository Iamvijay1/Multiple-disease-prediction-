# Multiple Disease Prediction System
A comprehensive web-based application designed to predict various health conditions based on user-input symptoms and medical parameters. This project leverages Machine Learning models integrated into a Django web framework to provide a seamless and interactive user experience.
## 🚀 Features
 * **Multi-Disease Support:** Predicts several diseases such as Diabetes, Heart Disease, Kidney Disease, and more (depending on the models integrated).
 * **User Authentication:** Secure login and registration system for users to manage their health data.
 * **Admin Dashboard:** Dedicated administrative interface to manage users and view system statistics.
 * **Responsive UI:** A clean, user-friendly interface built with HTML5, CSS3, and JavaScript.
 * **Real-time Prediction:** Instant results based on clinical parameters provided by the user.
## 🛠️ Tech Stack
 * **Backend:** Django (Python)
 * **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
 * **Database:** SQLite (Default Django DB)
 * **Machine Learning:** Python (Scikit-learn, Pandas, NumPy)
 * **Styling:** Custom CSS and Static files

## ⚙️ Installation & Setup
 1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Iamvijay1/Multiple-disease-prediction-.git
   cd Multiple-disease-prediction-
   
   ```
 2. **Create a Virtual Environment (Recommended):**
   ```bash
   python -m venv venv
   # Activate on Windows:
   venv\Scripts\activate
   # Activate on macOS/Linux:
   source venv/bin/activate
   
   ```
 3. **Install Dependencies:**
   *(Ensure you have a requirements.txt file or install Django and ML libraries manually)*
   ```bash
   pip install django pandas numpy scikit-learn
   
   ```
 4. **Perform Database Migrations:**
   ```bash
   python manage.py makemigrations
   python manage.py migrate
   
   ```
 5. **Run the Development Server:**
   ```bash
   python manage.py runserver
   
   ```
 6. **Access the App:**
   Open your browser and go to http://127.0.0.1:8000/.
## 🖥️ Usage
 1. **Register/Login:** Create a new account or log in with existing credentials.
 2. **Select Disease:** Choose the specific disease prediction model you wish to use from the dashboard.
 3. **Enter Data:** Input the required clinical parameters (e.g., Glucose levels, Blood Pressure, BMI).
 4. **Predict:** Click the "Predict" button to see the result generated by the Machine Learning model.
